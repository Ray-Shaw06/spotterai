    The method

    Generate, then audit

    The interesting part isn't the AI writing a workout — anyone can do that. It's the second system that checks the first one's work.

    1. 1

      You describe your training

      Goal, experience, days, equipment, session length, and any injuries — captured in a fast, fully custom form.

    2. 2

      Gemini drafts a program

      A serverless function (holding the API key) prompts Gemini for a strict-JSON weekly plan, validating and retrying on malformed output.

    3. 3

      Code audits the plan

      A pure-code evaluator estimates volume, balance, recovery, and injury risk — then scores the plan 0–100 and explains every flag.

    What the evaluator checks

    • Recovery — flags a week with no rest day.
    • Weekly volume — estimates sets per muscle group; flags overtraining or under-stimulation for the goal.
    • Push / pull balance — flags lopsided programming (e.g. all push, no pull).
    • Injury contraindications — maps stated injuries to risky movements and suggests regressions.
    • Beginner load sanity — flags advanced intensity or volume prescribed to a beginner.
    • Goal fit — checks rep ranges and structure against the stated goal.
