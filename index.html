      The method

      Generate, then audit

      The interesting part isn't the AI writing a workout — anyone can do that. It's the second system that checks the first one's work.

      1. 1

        You describe your training

        Goal, experience, days, equipment, session length, and any injuries — captured in a fast, fully custom form.

      2. 2

        Gemini drafts a program

        A serverless function (holding the API key) prompts Gemini for a strict-JSON weekly plan, validating and retrying on malformed output.

      3. 3

        Code audits the plan

        A pure-code evaluator estimates volume, balance, recovery, and injury risk — then scores the plan 0–100 and explains every flag.

      What the evaluator checks

      • Recovery — flags a week with no rest day.
      • Weekly volume — estimates sets per muscle group; flags overtraining or under-stimulation for the goal.
      • Push / pull balance — flags lopsided programming (e.g. all push, no pull).
      • Injury contraindications — maps stated injuries to risky movements and suggests regressions.
      • Beginner load sanity — flags advanced intensity or volume prescribed to a beginner.
      • Goal fit — checks rep ranges and structure against the stated goal.

        How SpotterAI audits plans

        Safety Lab

        SpotterAI does not blindly trust AI-generated plans. Every plan is checked by a deterministic safety and quality evaluator that looks for common programming problems like poor recovery, excessive volume, push/pull imbalance, goal mismatch, and conflicts with user limitations.

        This is not a medical certification or guarantee of safety. It is a transparent set of guardrails designed to catch obvious issues before a user follows a plan.

        Live red-team report

        The evaluator run against a battery of good and intentionally-bad plans. Each case lists what it should catch and whether it did — this exact suite also runs in CI, so the auditor can't silently regress.

        On-device · beta

        Real-time form check

        The plan auditor's physical-world twin: your webcam plus on-device pose estimation gives live form cues and counts reps — all in your browser. The video never leaves your device.

        What it checks

        • Auto rep counting from 3D joint angles, smoothed to cut jitter.
        • Form cues for squat, push-up, lunge, overhead press, biceps curl, RDL/hinge, and hip thrust.
        • “Other” counts reps for any movement (no form cues).

        Heuristic cues from a single 2D camera — not a coach or physiotherapist. It can't see your spine, load, or true 3D depth, so it can be wrong. Treat it as a mirror, not a judge; stop if anything hurts.
